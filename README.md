# CVProject

## :memo: Description
CVProject is a SPA project, inspired by https://neuethemes.net/preview/gridus/v2.2.3. 

#### :white_check_mark: Futures: 

    - Home page with picture, personal info and skills

    - Work experience page

    - Certificate page
    - Portfolio page
